# Historia de LINQ

Antes de LINQ los desarrolladores utilizaban:

- SQL para bases de datos
- XPath para XML
- foreach para colecciones

Esto obligaba a aprender múltiples lenguajes de consulta.

## Aparición

LINQ apareció en:

- C# 3.0
- .NET Framework 3.5

## Objetivo

Permitir consultas sobre cualquier origen de datos utilizando una única sintaxis.
